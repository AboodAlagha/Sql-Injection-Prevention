
# SQL Injection Prevention Using Prepared Statements

## Project Description
This project demonstrates how SQL Injection attacks occur and how they can be prevented using Prepared Statements in PHP.

## Features
- Search form using PHP
- Normal search example
- SQL Injection attempt example
- Secure database querying using Prepared Statements

## Screenshots
All required screenshots are included in the screenshots folder.

## Technologies Used
- PHP
- MySQL
- HTML

## Security
The project uses prepared statements to prevent SQL Injection attacks.
